# rpi_car_maze

창업연계공학설계입문 과목 5차 과제인 미로 탈출을 위한 코드입니다.

Python2를 사용하였고, 코딩과 테스트는 혼자 진행하였습니다.

### 사용법

``` 
git clone https://github.com/CertainlyT/rpi_car_maze.git

cd rpi_car_maze

python start_maze_solve.py
```
